### Текст задачи

необходимо заполнить случайными данными, заполнение организовать отдельным методом.
В задании 2 в текстовом файле хранятся целые числа по несколько в строке, исходный файл
необходимо заполнить случайными данными, заполнение организовать отдельным методом.
В задании 3 в текстовом файле хранится текст.
В задании 4 бинарные файлы, содержат числовые данные, исходный файл необходимо заполнить
случайными данными, заполнение организовать отдельным методом.
В задании 5 бинарные файлы содержат величины типа struct, заполнение исходного файла
необходимо организовать отдельным методом, обязательно использовать xml сериализацию,
решение без сериализации не принимается.
В задании 10 исходные данные хранятся в текстовом файле

### Алгоритм решения

1. Запуск программы 
2. Выводится приглашение "Введите first (true/false): ", пользователь вводит строку, которая преобразуется в тип bool через bool.Parse() и сохраняется в переменную a.
3. Выводится приглашение "Введите second (true/false): ", пользователь вводит строку, которая преобразуется в bool и сохраняется в переменную b.
4. Ввызывается конструктор LogicalPair(a, b), который сохраняет переданные значения в приватные поля first и second, создается объект obj.
5. Вызывается метод ToString() для obj, который возвращает строку "First = X, Second = Y"
6. Вызывается метод Implication() для obj, вычисляет логическое выражение !first || second
7. Вызывается конструктор копирования LogicalPair(obj), который копирует значения полей first и second из переданного объекта, создается объект copy.
8. Вызывается метод ToString() для copy и результат выводится на экран.
9. Выводится приглашение "Введите имя: ", пользователь вводит строку, которая сохраняется в переменную name.
10. Вызывается конструктор LogicalExpression(a, b, name), который через base(a, b) вызывает конструктор родительского класса для инициализации полей first и second, затем сохраняет переданное имя в поле name, создается объект expr.
11. Вызывается переопределенный метод ToString() для expr, который возвращает строку "Name = X, First = Y, Second = Z", результат выводится на экран.
12. Вызывается метод IsEqual() для expr, который сравнивает значения first и second через GetFirst() == GetSecond(), результат выводится на экран.
13. Вызывается метод Invert() для expr, который меняет значения полей на противоположные через SetValues(!GetFirst(), !GetSecond()).
14. Вызывается метод ToString() для expr и результат выводится на экран.
15. Вызывается конструктор копирования LogicalExpression(expr), который через base(other) копирует поля родительского класса (first и second), затем копирует поле name, создается объект exprCopy.
16. Вызывается метод ToString() для exprCopy и результат выводится на экран.
